# SlardarOS
>Just a toy OS

## Features:
* support STL container
* support C++14
* bootloader
* virtual memory
* interruption
* privilege level switch
* mutil-processes (in progress)

## Dependency:
* nasm >= 0.98
* qemu >= 2.9.0
* gcc >= 4.8.1 (cross-compiler)
* OS X EL Capitan (only support os x for now)

## Usage:
* `make qemu` or
* `make debug`
